# TradeHub — Roadmap

## Phase 1 — Foundation (Week 1)
**Goal:** Live directory with first clients, deployed and indexable.

- [x] Project scaffold (Next.js 14, Tailwind, Vercel)
- [ ] Homepage with hero, search, featured trades, trust messaging
- [ ] Category pages for top 5 trades (plumbers, electricians, roofers, builders, landscapers)
- [ ] Location pages for top 3 areas (Solihull, Birmingham, Redditch)
- [ ] Combined category+location pages (SEO money pages)
- [ ] Individual trade profile pages
- [ ] First 5–10 client listings added
- [ ] JSON-LD schema on all pages
- [ ] llms.txt, robots.txt, sitemap.xml
- [ ] Deploy to Vercel, custom domain
- [ ] Submit to Google Search Console

## Phase 2 — Content & Coverage (Week 2–3)
**Goal:** More pages, more locations, content marketing starts.

- [ ] Blog section live with 3–5 articles
  - "How to find a reliable plumber in Birmingham"
  - "Questions to ask before hiring an electrician"
  - "What does a good roofer quote look like?"
- [ ] Expand to 8–10 location pages
- [ ] Add more trade categories (plasterers, painters, carpenters, locksmiths)
- [ ] "List your business" CTA page
- [ ] About page
- [ ] Google Search Console monitoring — fix any crawl issues
- [ ] Start link building (local directories, trade associations)

## Phase 3 — Lead Gen & Monetisation (Month 2)
**Goal:** Test revenue streams, prove the model.

- [ ] Featured listings live (£50/mo upsell to existing clients)
- [ ] Lead gen test for 1–2 high-value trades (kitchen fitters, bathroom installers)
  - Dedicated landing pages
  - Contact form with email routing
  - Track conversions
- [ ] Google Ads campaign for 1 client (proof of concept)
- [ ] First referral program sign-ups
- [ ] 15–20 total listings

## Phase 4 — Scale & Automate (Month 3+)
**Goal:** Reduce manual work, grow client base, expand geography.

- [ ] Admin panel for managing listings (add/edit/remove without code)
- [ ] Analytics dashboard — page views, form submissions, calls per client
- [ ] Contact form tracking with email notifications
- [ ] Automated invoice/billing integration
- [ ] Expand to neighbouring areas (Coventry, Warwick, Stratford)
- [ ] Client portal: tradespeople can update their own profile
- [ ] Review collection system (automated email/SMS after job)
- [ ] 50+ listings target
- [ ] Monthly revenue target: £3,000+ MRR
